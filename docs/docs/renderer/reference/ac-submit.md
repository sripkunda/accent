# ac-submit

`ac-submit` is simply an extension of the `ac-on` and serves as an alias to the `ac-on *onsubmit` identifier.

### Usage

**Using `ac-submit`:**

```html
<form ac-submit="console.log('Submtited!');"></form>
```

**`ac-on` Equivalent:**

```html
<form ac-on *onsubmit="console.log('Submtited!');"></form>
```
